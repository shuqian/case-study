自定义模板引擎目录
	mytpl.class.php		自己定义模板引擎类MyTpl所在文件
	index.php		使用自定义模板引擎类的php文件
	templates		自定义的模板保存目录
		header.html	头部模板文件
		main.html	主模板文件
		footer.html	尾部模板文件

Smarty目录
	init.inc.php		Smarty初使化文件
	project目录		第一个Smarty实例演示
	function目录		模板中的函数应用演示
	config目录		在模板中使用配置文件
	var目录			在模板中使用保留变量
	modify目录		使用变量调解器的应用
